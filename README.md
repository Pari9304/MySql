# SQL Portfolio Projects

## Overview

This repository contains a collection of SQL projects and practice exercises demonstrating fundamental to advanced SQL concepts. The projects focus on database management, data analysis, business reporting, joins, subqueries, and pattern matching techniques commonly used in Data Analyst and Business Analyst roles.

## Skills Demonstrated

* Database Creation and Management
* CRUD Operations (Create, Read, Update, Delete)
* Aggregate Functions
* Data Filtering
* Sorting and Ranking
* Pattern Matching using LIKE and REGEXP
* Joins (INNER, LEFT, RIGHT, SELF JOIN)
* Subqueries
* Grouping and Reporting
* Business Analytics Queries
* KPI Analysis

## Tools Used

* MySQL
* SQL
